HRK'S AI API DOCS

GET /openai

Description: Get a response from the GPT-4 model.

Parameters:

  • gpt4 (required): Input text for the GPT-4 model.

Example:

GET /openai?gpt4=Hello

Response:

{"response": "Hi! How can i assist you today?"}

GET /anthropic

Description: Get a response from the Claude v2 model.

Parameters:

  • claudeV2 (required): Input text for the Claude v2 model.

Example:

GET /anthropic?claudev2=Hi.

Response:

{"response": "Hello!"}
