HRK'S AI API DOCS

GET /openai

Description: Get a response from the GPT-4 model.

Parameters:

  • gpt4 (required): Input text for the GPT-4 model.

Example:

GET /openai?gpt4=Hello

Response:

{"response": "Hi! How can i assist you today?"}

Python Integration:


import requests

url = 'https://green-devil.tech/openai'
params = {'gpt4': 'Hello'}

response = requests.get(url, params=params)
print(response.json())

cURL Integration:


curl -X GET 'https://green-devil.tech/openai?gpt4=Hello'

Next.js Integration:


import fetch from 'isomorphic-fetch';

async function fetchData() {
  const response = await fetch('https://green-devil.tech/openai?gpt4=Hello');
  const data = await response.json();
  console.log(data);
}

fetchData();
