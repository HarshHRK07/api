API Documentation

GET /openai

Description: Get a response from the GPT-4 model.

Parameters:

  • gpt4 (required): Input text for the GPT-4 model.

Example:

GET /openai?gpt4=How%20are%20you%3F

Response:

{"response": "GPT-4 response text"}

GET /anthropic

Description: Get a response from the Claude v2 model.

Parameters:

  • claudev2 (required): Input text for the Claude v2 model.

Example:

GET /anthropic?claudev2=Tell%20me%20a%20joke.

Response:

{"response": "Claude v2 response text"}
