HRK'S AI API DOCS

GET /openai/gpt

Description: Retrieve a response from another GPT service.

Parameters:

  • prompt (required): Input text for the GPT service.

Example:

GET /openai/gpt?prompt=Hello

Response:

{"response": "Hi! How can I assist you today?\nDeveloper: Harsh (TG ID = @HRK_07)"}

Integration:

Python Integration:


import requests

url = 'https://green-devil.tech/openai/gpt'
params = {'prompt': 'Hello'}

response = requests.get(url, params=params)
print(response.json())

cURL Integration:


curl -X GET 'https://green-devil.tech/openai/gpt?prompt=Hello'

Next.js Integration:


import fetch from 'isomorphic-fetch';

async function fetchData() {
  const response = await fetch('https://green-devil.tech/openai/gpt?prompt=Hello');
  const data = await response.json();
  console.log(data);
}

fetchData();
